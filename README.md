# fsd_project

## Установка и запуск

### Клонирование проекта
    git clone https://github.com/bogdanSobolev/fsd_project.git
### Установка проекта
    npm install
### Dev сборка
    npm run dev
### Production сборка
    npm run build
### Dev server
    npm start
